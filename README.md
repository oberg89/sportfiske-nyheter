# Sportfiske Nyheter

En modern nyhetssajt om sportfiske i Sverige, byggd med endast HTML, CSS och JavaScript. Projektet innehåller både en startsida med nyhetsflöde och en sidvisning för full artikel med kommentarer, likes och dislikes.

## Funktioner

- Skapa nya artiklar via formulär.
- Radera artiklar direkt i nyhetsflödet.
- Visa toast-notifieringar vid skapande och radering.
- Spara artiklar i `localStorage` så de behålls efter omladdning.
- Se en fullständig enskild artikel på `article.html`.
- Kommentera kommentarsfält på artikel-sidan.
- Likes och dislikes som sparas i `localStorage`.
- Responsiv layout med Bootstrap 5 för mobil och desktop.

## Hur du använder projektet

1. Öppna `index.html` i din webbläsare.
2. Skapa nya artiklar med formuläret.
3. Klicka på en artikel för att se hela innehållet.
4. Radera artiklar med "Radera"-knappen.
5. Lägg till kommentarer och gilla/ogilla på artikel-sidan.

## Teknik

- HTML
- CSS
- JavaScript
- Bootstrap 5 (CDN)
- Bootstrap Icons (CDN)
- `localStorage` för att spara artiklar, kommentarer och reaktioner

## Projektstruktur

- `index.html` — Startsida med nyhetsflöde och artikelhantering.
- `article.html` — Visar vald artikel i helvy med kommentarer och likes/dislikes.
- `script.js` — All logik för artiklar, localStorage, toast-notiser, kommentarer och reaktioner.
- `styles.css` — Anpassad styling för sajten.
- Bildfiler — Illustrerar olika sportfiske-nyheter.

## Viktigt

- Projektet använder endast HTML, CSS och JavaScript.
- Inga JavaScript-ramverk används.
- CSS-ramverk (Bootstrap) används för layout och design.


## Starta projektet lokalt

Det räcker att öppna `index.html` i valfri webbläsare.
